# Taller-RMI-Sistemas-Distribuidos-Cliente-Servidor-con-Java-RMI
Taller académico de la asignatura Introducción a los Sistemas Distribuidos ( 2026 - 01), en el que se implementa un sistema cliente-servidor distribuido utilizando Java RMI (Remote Method Invocation). El objetivo principal es demostrar la comunicación entre dos máquinas virtuales a través de la invocación de métodos remotos.
